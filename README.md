有关我的Dotfiles的配置文件仓库

- `shell`相关配置(`fish`、`starship`)

## 安装
你可以通过`stow`符号管理器来安装对应内容 也可以使用我的安装脚本

### Termux
```shell
git clone https://github.com/Sugars0612/dotfiles
cd dotfiles
chmod +x ./bootstarp-termux.sh ./imstall-termux.sh
./bootstarp-termux.sh
./install-termux.sh
```
